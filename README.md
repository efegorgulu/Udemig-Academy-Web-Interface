# Udemig Akademy Web Interface (HTML, SCSS & CSS)

A static **academy web interface** built with **HTML5, SCSS (SASS), and CSS3**, featuring a modern landing page, course sections, informational content, and a structured layout compiled from modular SCSS files.

## Screenshot
![](Screen.gif)

## Highlights
- **Header layout** with brand logo, navigation menu, and social media icons  
- **Responsive navigation design** with a mobile menu button UI  
- **Hero (Home) section** with headline, description, and call-to-action button  
- **Courses section** showcasing different learning categories with images  
- **About Us section** with detailed informational content and image layout  
- **Popular Courses section** with course cards including duration and pricing  
- **Contact form UI** with input fields and textarea for user interaction (**visual only — no functional logic**)  
- **Footer section** with multiple columns (services, about, help, social links)  
- **SCSS modular structure** using partial files (`_header.scss`, `_home.scss`, etc.)  
- **Use of variables and mixins** for reusable and maintainable styling  

## Technologies Used
- **HTML5**
- **SCSS (SASS)**
- **CSS3**
- **Font Awesome** for UI icons  

## Installation
1. Clone or download this repository  
2. Open `index.html` in your preferred web browser  

## Project Notes
- Front-end user interface only (**no backend or JavaScript functionality**)  
- SCSS is structured using **partials and compiled into a single CSS file**  
- Styling uses **variables, nesting, and modular architecture**  
- Images are loaded locally from the `images/` directory  
- Navigation links and contact form are **UI-only**
